## Install

First, clone this repo
```
git clone https://github.com/Ackerr/dotfiles.git && cd dotfiles
```

Then, run the shell script

```
bash ./install.sh
```
> You can find original configuration in `~/.backup`, and remember replace `.gitconfig` email to yours
